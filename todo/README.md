# 기능 목록

## 잔돈 생성 기능

> 투입 금액으로 잔돈을 생성한다

- 무작위로 생성

## 잔돈 반환 기능

> 조건에 맞추어 잔돈을 반환한다

- 남은 금액이 상품의 최저 가격보다 적거나, 모든 상품이 소진된 경우 바로 잔돈을 돌려준다.

- 잔돈을 반환할 수 없는 경우 잔돈으로 반환할 수 있는 금액만 반환한다.

## 금액 계산 기능

> 투입 금액을 계산한다

- 구매한 상품의 가격만큼 빼기

## 예외 처리

1. 자판기 보유 금액 예외

2. 상품 가격, 수량, 상품명 예외

3. 구매한 상품명 예외
